Yarn - Как начать работу?
1).Сначала установим глобальный Yarn: npm install -g yarn.
2).Проверка версии: yarn -v.
3).Изменим версию Yarn для конкретной папки: yarn set version berry (То есть Yarn 2).
--После выполнения данной команды установка будет завершена, и можно переходить к установке зависимостей!

-Добавление зависимостей:
----------------------------

0. yarn init? — ?инициализация проекта.
----------------------------------------------------------------

1. Установить пакеты - yarn add [package name]
----------------------------------------------------------------

2. yarn add <package> [--dev]? —? добавление пакета
--Добавление зависимости:
yarn add [package]
yarn add [package]@[version]
yarn add [package]@[tag]

----------------------------

--Добавление зависимости к различным категориям зависимостей:
yarn add [package] --dev
yarn add [package] --peer
yarn add [package] --optional
----------------------------------------------------------------

3. yarn remove <package>? — ?удаление пакета
----------------------------------------------------------------

4. yarn up <package>? —? обновление пакета
yarn upgrade [package name]
yarn upgrade [package]@[version]
yarn upgrade [package]@[tag]
----------------------------------------------------------------

5. Удаление зависимости - yarn remove [package name]
----------------------------------------------------------------

6. Менеджер обновлений - yarn upgrade
----------------------------------------------------------------

7. Запустить скрипт - yarn run
----------------------------------------------------------------

8. Проверить наличие устаревших пакетов - yarn outdated
----------------------------------------------------------------

9. Запустить тесты для текущего пакета - yarn test
----------------------------------------------------------------
